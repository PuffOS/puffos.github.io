Welcome to universe of PuffOS! This disstro is based on Debian SID

The PuffOS project aims to provide pure Linux experience by freeing Debian from bloats. PuffOS is based on Debian's installation with Deboostap and uses Debian's SID repos. (Note that it won't be as robust as Debian Buster for stability.)

Inside the PuffOS iso file

  • An installer (17g) that allows you to install the system comfortably
  • GNU/Linux Debian Base System
  • Up-to-date Gnome Desktop environment and applications that allow you to do your general work easily
  • LightDM Desktop Manager
  • PuffOS project official artworks

There are. All software other than these require manual installation.
